todoking

I wrote this software during my second year at university and it was created for the purpose of reducing the clutter on my desk. Basic access control logic lets users create their own profiles and add todo items to a list. These items can then be deleted, marked as completed or abandoned. Backgrounds can be set and items can be displayed according to a custom filter.

lichessuseractivity

This project came to me one day as I was watching a live game of chess on lichess.org. I wondered if there was a way I could track whether a player was online or not so that I could watcht the games live. This app enables the user to do just that: be informed of whether a user is online or not.
