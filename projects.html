onechess

May. 2021

As a part of my curriculum I developed a locally dynamic website using NodeJS and PostgreSQL with a custom modular chess engine that is used across the variants (international, xiangqi and shogi). This chess engine can be modularized and I wish to do so by creating a public js library named chesscreator for others to create their own variants.

todoking

Jan. 2020

I wrote this software during my second year at university and it was created for the purpose of reducing the clutter on my desk. Basic access control logic lets users create their own profiles and add todo items to a list. These items can then be deleted, marked as completed or abandoned. Backgrounds can be set and items can be displayed according to a custom filter.
